To use this module, add the following configuration block to the modules array in the `config/config.js` file:
```js
var config = {
    modules: [
        {
            module: 'MMM-Nixie-Clock',
            position:"top_left",
            size: 75,            ** CSS automatically sizes the image size - 10%
        }
    ]
}
```
